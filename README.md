# Telegram-бот "Кулинарная книга"

Бот для хранения и поиска рецептов. 

### Технологии:
- Python 3.12
- SQLAlchemy (SQLite)
- python-telegram-bot v20+
- Unittest

### Как запустить:
1. Установите зависимости: `pip install -r requirements.txt`
3. Настройте токен в `.env`
   Пример:
   Создайте в папке файл .env.
   Напишите в нем:
   BOT_TOKEN="ВАШ ТОКЕН"
5. Запустите: `python main.py`
