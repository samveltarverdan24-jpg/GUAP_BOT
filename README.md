# Telegram-бот "Кулинарная книга"

Бот для хранения и поиска рецептов. 

### Технологии:
- Python 3.12
- SQLAlchemy (SQLite)
- python-telegram-bot v20+
- Unittest

### Как запустить:
1. Установите зависимости: `pip install -r requirements.txt`
2. Настройте токен в `.env`
3. Запустите: `python main.py`
